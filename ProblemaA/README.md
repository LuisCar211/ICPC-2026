# Problem A - Anxiety at the restaurant

## Idea de solución

El problema pide verificar si el dinero reunido por el grupo es suficiente para pagar:

- el total de la cuenta
- más un 10% de propina

Primero se suman todos los precios de los artículos para obtener el total de la cuenta.

Después se calcula cuánto dinero se necesita considerando el 10% extra de propina.

Finalmente se suman todas las cantidades aportadas por las personas y se compara:

- Si el dinero reunido es mayor o igual al total con propina, se imprime `YES`.
- En caso contrario, se imprime `NO`.
